# TodoApp - Mobile

Aplicacao responsavel por cadastrar todos.

Entre as funcionalidades estão:
 - Criar um novo TODO
 - Marcar um TODO como done
 - Excluir um TODO
 
Para a estilizacao foi utilizado o StyleSheets do proprio React Native

```bash
  npm install
  expo start
```
